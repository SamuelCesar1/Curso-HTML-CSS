Para adicionar "<" e ">" no seu texto para representar a escrita de uma tag, você precisa digitar "&", seguido de "lt" e finalizando com ";". Já para o fechamento do simbolo, basta repetir o "&"" seguido de "gt" e finalizando com ";".
Assim você irá conseguir representar a tag, sem que o compilador entenda que está abrindo uma estrutura.

Para quebrar a linha dentro de um mesmo
parágrafo, basta usar a tag < br >.

  1. 🖖
  2. 🤓
  3. 😎

Estes são alguns dos ecemplos de uma lista muito extensa que pode ser utilizada em seu site. Para implementar um emoji, é muito simples, você vai usar o "&" o "#" seguido da letra "X".
Exemplo:&#x.
Após essa sequencia, deve ser colocado o código do emoji, e no final acrescentar ";".

Imagens

Sempre que for utilizar uma imagem em seu site, tome muito cuidado com a questão dos direito autorais das imagens utilizadas. Procure em banco de dados de imagens online que te deem os direitos de personalizar, editar e comercializar a imagem. Vou listar alguns desses sites que te oferecem esse tipo de imagem, mas lembrese, alguns deles será necessário pesquisar em ingles. Caso tenha alguma dificuldade para pesquisar, experimente usar o google tradutor para te auxiliar.

  • UnSplash
  • Pexels
  • FreePik
  • Rawpixel
  • Pixabay
  • Libreshot

Os principais modelos de imagens utilizadas hoje na internet, é o JPEG e o PNG. os JPEGs têm menos dados que os PNGs e, portanto, costumam ser menores. Ao contrário dos arquivos JPEG, o formato PNG é compatível com fundos transparentes, tornando-o ideal para design gráfico.
Com a popularização das Figurinhas do WhatsApp, o modelo GIF voltou a ser bastante utilizado também, mas os dominantes ainda são o JPEG e o PNG.

Hierarquia de Títulos

Dentro dos títulos, existe uma hierarquia de deve ser respeitada para que o desenvolvimento do seu projeto seja bem organizado. Existem 6 tipos de títulos, que vai do h1 ao h6 sendo títulos e sub-títulos contidos um dentro do outro. Ficou confuso? Então espera que eu vou te explicar de uma forma um pouco melhor. Acompanhe a imagem a seguir:

Esta é uma imagem de índice de um manual de execução de estruturas metálicas que eu peguei no Yumpu.

Você conseguiu notar na imagem a seguir uma certa hierarquia?
Temos na imagem os números 1,2,3 e 4 acompanhados pelos seus títulos. Introdução, Orçamentação, Gestão de Produção e Projecto e Detalhes. Esses são os títulos principais, que dentro da estrutura ou da hhierarquia, são representados pela tag "h1".

Vamos pegar como base agora o título de número 2 - Orçamentação. Você percebeu que ele tem em sua estrutura alguns subtitulos que estão representados pela númeração: 2.1 até o 2.6, estes são sub-títulos do título principal e podem ser representados por h2 ou seja, são sub-títulos de Orçamentação, assim funciona também na hierarquia dos títulos, o h2 é um sub-título de h1. Podemos então dizer que o h2 está contido dentro do h1.

Se pararmos para observar mais um pouco, veremos também que existe outra estrutura com a númeração: 2.4.1 até 2.4.4, que tabém é um sub-título, estes são representados por h3 pois são sub-títulos do h2. Se ficou um pouco confuso, vou explicar melhor. Este é o nosso h2 Representado por 2.4 - Distribuição Percentual de Custos para Unidades Industriais. E este é um dos exemplos dos nossos h3 Representado por 2.4.1 - Descrição geral dos tipos de pavilhões.

Agora eu acredito que ficou um pouco mais fácil para identificarmos as nossas duas estruturas de h4. São elas 2.4.2.1 e 2.4.2.2, são sub-títulos da 2.4.2
